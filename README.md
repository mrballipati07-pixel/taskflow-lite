TaskFlow Lite

TaskFlow Lite is a modern task management web application built using HTML, CSS, and JavaScript. It helps users organize daily tasks, track progress, manage deadlines, and improve productivity through a clean and responsive interface.

Features
•	Add new tasks
•	Edit existing tasks
•	Delete tasks
•	Mark tasks as completed or active
•	Search tasks instantly
•	Due date support
•	Progress tracking with a progress bar
•	Dark mode support
•	Local Storage persistence
•	Responsive design for mobile and desktop
•	Smooth animations and modern UI

Technologies Used
•	HTML5
•	CSS3
•	JavaScript (ES6 Modules)
•	Local Storage API

Installation
1.	Clone the repository:
    git clone https://github.com/mrballipati07-pixel/taskflow-lite.git
2.	Open the project folder in Visual Studio Code.
3.	Install the Live Server extension.
4.	Right-click on index.html and select "Open with Live Server".

Usage
1.	Enter a task in the input field.
2.	Select a due date if needed.
3.	Click "Add Task".
4.	Use the Done button to mark tasks as completed.
5.	Use the Edit button to modify tasks.
6.	Use the Delete button to remove tasks.
7.	Search tasks using the search bar.
8.	Toggle Dark Mode for a different theme.
